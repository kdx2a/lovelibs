# lovelibs
A small LÃ–VE 2D utility that downloads and 'update' common libs. Small personnal project, don't take it too seriously.
Require Lua 5.1 or superior and `wget`.

# usage
List all available libs  
`lovelibs list all`

Install lib(s)  
`lovelibs install <lib(s)>`

Remove lib(s)  
`lovelibs remove <lib(s)>`

List installed libs  
`lovelibs list`

Update libs  
`lovelibs update`
