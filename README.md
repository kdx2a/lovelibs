# lovelibs
A small LÃ–VE 2D utility that downloads and 'update' common libs. Small personnal project, don't take it too seriously.
Require Lua 5.1 or superior.

# usage
Install a lib (see list of libs bellow)  
`lovelibs install <package>`

Insatll multiple libs  
`lovelibs install <package> <package> <package> ...`

Update libs  
`lovelibs update`
